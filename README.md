# YouBike 站點地圖

這是一個使用 Vue 3 + Vite 開發的 YouBike 站點地圖應用，整合了 OpenStreetMap 和 YouBike 即時站點資訊。

## 功能特點

- 即時顯示 YouBike 站點位置和可用車輛數
- 站點群聚顯示，優化大量站點的展示
- 站點搜索功能
- 附近站點查詢
- 自適應地圖標記顏色（根據可用車輛數）
- 響應式設計

## 技術棧

- Vue 3
- TypeScript
- Vite
- Leaflet
- Pinia
- OpenStreetMap

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 構建生產版本
npm run build
```
